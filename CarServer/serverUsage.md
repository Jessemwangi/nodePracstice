# Carserver Usage

## get all cars

```
localhost:3000/cars
```

## get all types
```
localhost:3000/carstypes
```

## serch by licence
```
localhost:3000/search/bylicense?value=abc-1
```
## serch by model
```
localhost:3000/search/bymodel?value=hoppa
```

# Version 2:

## search by licence
```
localhost:3000/search?key=licence&value=abc-1
```
## search by model
```
localhost:3000/search?key=model&value=hoppa
```
